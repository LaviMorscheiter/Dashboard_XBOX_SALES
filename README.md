## 📌 Sobre o Projeto

Este projeto foi desenvolvido com foco em **análise de dados e construção de dashboards** utilizando uma base de dados fornecida pela plataforma **DIO (Digital Innovation One)**.

O objetivo principal foi transformar dados brutos relacionados a assinaturas do **Xbox Game Pass** em informações visuais relevantes, facilitando a interpretação dos dados e auxiliando na tomada de decisão.

O dashboard apresenta indicadores sobre:

* 💰 Receita total gerada
* 🎮 Season Pass adquiridos

---

## 🗂️ Estrutura da Base de Dados

A base utilizada contém informações fictícias sobre usuários e assinaturas do Xbox Game Pass.

### 📑 Principais colunas analisadas

| Coluna                  | Descrição                                      |
| ----------------------- | ---------------------------------------------- |
| `Name`                  | Nome do assinante                              |
| `Plan`                  | Plano contratado                               |
| `Start Date`            | Data de início da assinatura                   |
| `Auto Renewal`          | Indica renovação automática                    |
| `Subscription Price`    | Valor da assinatura                            |
| `Subscription Type`     | Tipo da assinatura (Mensal, Trimestral, Anual) |
| `EA Play Season Pass`   | Possui EA Play                                 |
| `Minecraft Season Pass` | Possui Minecraft Pass                          |
| `Coupon Value`          | Valor de cupom aplicado                        |
| `Total Value`           | Valor total gasto                              |

## 📈 Análises Desenvolvidas

Durante o projeto foram realizadas:

* Limpeza e organização da base de dados
* Criação de tabelas auxiliares
* Construção de métricas e indicadores
* Consolidação de dados para análise visual
* Desenvolvimento de dashboard interativo

### 📊 Métricas analisadas

* Total arrecadado
* Quantidade de assinantes
* Distribuição por tipo de assinatura
* Impacto de cupons no valor final
* Relação entre renovação automática e faturamento
* Venda de Season Pass adicionais
